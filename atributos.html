ATRIBUTOS DE ESTILO
Atributo Visualizacion Descripcion
El atributo color en CSS se utiliza para establecer el color del texto dentro de un elemento HTML.
Puedes especificar el color utilizando palabras clave (como red), valores hexadecimales (como #FF0000), valores RGB (como rgb(255, 0, 0)).
El atributo font-size en CSS define el tamaño del texto en un elemento HTML. Puedes especificar el tamaño
usando unidades como píxeles (px), em, rem, puntos (pt), porcentajes (%), o palabras clave como small, medium, large.
El atributo font-family en CSS especifica la fuente o tipo de letra que se utilizará para el texto dentro de un
elemento HTML. Puedes listar varias fuentes como opciones, separadas por comas, donde el navegador utilizará la
primera fuente disponible en el sistema del usuario.
El atributo font-weight en CSS define el grosor de la fuente de un texto, permitiendo valores desde 100 (muy delgado)
hasta 900 (muy grueso), o mediante palabras clave como normal, bold, bolder, y lighter.
El atributo text-decoration en CSS aplica efectos decorativos al texto, como subrayado (underline), tachado (line-through),
o sobrelineado (overline). También permite combinar múltiples decoraciones y definir su estilo y color.
El atributo padding en CSS define el espacio interior entre el contenido de un elemento y su borde, aplicando
un relleno en uno o varios lados del elemento.
El atributo border-radius en CSS redondea las esquinas de un elemento, permitiendo definir un radio específico
para cada esquina o para todas, creando bordes circulares o elípticos.
